
-------------------------------------------
PACKAGE ethuebung FOR LATEX EXERCISE SHEETS
-------------------------------------------

This is a LaTeX package meant for developing exercise sheets with
their solutions sheets at ETH Zurich. It takes care of compiling both
exercise and solution sheets, correct exercise numbering, with
exercise part numbering, inline solutions etc.
